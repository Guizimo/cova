## üöÄ Getting Started

Make sure you have [Node.js](https://nodejs.org/) (v18+ recommended) and [pnpm](https://pnpm.io/) installed in your development environment.

```bash
# Clone the repository
git clone https://github.com/guizimo/cova.git

# Enter the project directory
cd co'va

# Install dependencies
pnpm install

# Start the development server
pnpm dev
```
